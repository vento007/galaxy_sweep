## Requirements

- Flutter SDK
- Tested targets: `macos`, `chrome`, `windows`
- Untested targets: `android`, `ios`

## Install

```bash
flutter pub get
```

## Run

```bash
flutter run
```

Specific target examples:

```bash
flutter run -d macos --release
flutter run -d chrome --release
flutter run -d windows
```
